Setup

  1. Open Google AI Studio
  2. Navigate to the Speech section
  3. Return here, paste your JSON config, and click Start Automation

JSON Format

{
  "title": "Horror Story - Ep 1",
  "scene": "Late night. Dense forest. Old mansion.",
  "sampleContext": "Cinematic horror narration.",
  "clearExistingBlocks": true,
  "outputFilename": "horror_ep1",
  "speakers": [
    {
      "speakerIndex": 0,
      "aliasName": "Narrator",
      "voice": "Alnilam",
      "audioProfile": "Deep, resonant narrator.",
      "style": "Whisper",
      "pace": "Staccato",
      "accent": "Neutral"
    },
    {
      "speakerIndex": 1,
      "aliasName": "Aryan",
      "voice": "Alnilam",
      "audioProfile": "Scared young man.",
      "style": "Empathetic",
      "pace": "Natural",
      "accent": "Neutral"
    }
  ],
  "blocks": [
    { "speakerIndex": 0, "text": "हवेली गाँव से थोड़ा बाहर थी।" },
    { "speakerIndex": 1, "text": "मैंने camera ऑन किया।" }
  ]
}

Available Options

Style Vocal Smile, Newscaster, Whisper, Empathetic, Promo/Hype, Deadpan
Pace Natural, Rapid Fire, The Drift, Staccato
Accent Neutral, American (Gen), American (Valley), American (South), British (RP), British (Brixton), Transatlantic, Australian

Notes

  • speakerIndex in blocks references the speakers array (0-based).
  • Speaker settings are configured once per unique speakerIndex.
  • Set clearExistingBlocks: true to wipe the slate before starting.
  • The extension renames downloaded files using outputFilename.
  • Audio generation may take 30 s – 3 min depending on script length.
