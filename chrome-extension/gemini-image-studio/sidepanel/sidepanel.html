        Setup

        1. Open your provider: Gemini or ChatGPT
        2. Select the provider in Settings
        3. Start a chat with image generation enabled
        4. Paste your scene script, JSON, or CSV, validate, then click Start Queue

        Script format (recommended)

        Scene 1:
Characters Present: Aman, Neha, Sarita
Image Prompt:
Old three-floor ancestral house on Delhi outskirts…

Scene 2:
Characters Present: Aman
Image Prompt:
Empty corridor lit by flickering bulb…

        CSV format

        scene,characters_present,image_prompt
Scene 1,"Aman, Neha, Sarita","Old three-floor ancestral house…"
Scene 2,Aman,"Empty corridor lit by flickering bulb…"

        JSON format

        {
  "projectName": "horror_ep1",
  "scenes": [
    {
      "scene": "Scene 1",
      "characters_present": "Aman, Neha, Sarita",
      "image_prompt": "Old three-floor ancestral house…"
    }
  ]
}

        Studio (Phase 3)

        • Templates — wrap prompts with {{variables}}
        • Character library — auto-append appearance/style for named characters
        • Reference images — uploaded before each scene for consistency

        Export (Phase 2)

        • Images save to <project>/01_scene_name.png
        • Metadata JSON saves to <project>/Metadata/
        • ZIP bundle includes Images + Metadata + Prompts
        • Disable Chrome “Ask where to save each file” for smooth auto-download

        Providers

        • Gemini — gemini.google.com (image generation chat)
        • ChatGPT — chatgpt.com (GPT-4o / Images — ask it to create images)

        ChatGPT tips

        • Use a model that supports image generation (GPT-4o, Images)
        • Start a fresh chat before batch runs for best results
        • Prompts like “Create an image of …” work reliably

        Tips

        • Keep one Gemini tab open during batch runs
        • Use descriptive scene names for auto file naming
        • Use the Gallery tab to bulk-export an existing chat
