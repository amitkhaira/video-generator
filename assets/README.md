# Assets

Place your background music file here:

    assets/background_music.mp3

Any royalty-free MP3 will work. The script mixes it at low volume (~-18 dB)
underneath the narration. If this file is missing, the script proceeds with
narration-only audio.

Recommended sources for royalty-free music:
- https://pixabay.com/music/
- https://freesound.org/
- https://incompetech.com/music/
